关卡 6：平方的和与和的平方之差

前十个自然数的平方的和是

$1^2 + 2^2 + … + 10^2 = 385$

前十个自然数的和的平方是

$(1 + 2 + … + 10)^2 = 552 = 3025$

因此前十个自然数的平方的和与和的平方之差是 $3025 − 385 = 2640$。

求前一百个自然数的平方的和与和的平方之差。

$1^2+2^2+\cdots+x^2 = \frac{1}{6}x(x+1)(2x+1)$

$(1+2+\cdots+x)^2 = \left[\frac{(1 + x)x}{2}\right]^2$

$\left[\frac{(1 + x)x}{2}\right]^2 - \frac{1}{6}x(x+1)(2x+1) = \frac{x(x + 1)(x - 1)(3x + 2)}{12}$
